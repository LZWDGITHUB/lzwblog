```
 // 1、查看已经安装的node版本
nvm ls
 
// 2、查看可以安装的node版本
nvm ls available
 
// 3、安装指定版本
nvm install 版本号
 
// 4、切换到指定版本
nvm use 版本号
 
// 5、显示当前版本
nvm current
 
// 6、给不同的版本添加别名
nvm alias
 
// 7、删除已定义的别名
nvm unalias
 
// 8、在当前环境下， 重新全局安装指定版本号的npm包
nvm reinstall-packages
 
// 9、打开nodejs控制
nvm on
 
// 10、关闭nodejs控制
nvm off
 
// 11、查看设置代理
nvm proxy
 
// 12、卸载指定版本
nvm uninstall 版本号
 
// 13、设置和查看root路径
nvm root [path]
 
// 14、查看nvm版本
nvm version
```
